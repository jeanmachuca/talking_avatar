# Talking Avatar

A web-based talking avatar implementation that supports both English and Spanish languages. The avatar's mouth animates in sync with the speech, creating an interactive speaking experience.

## Features

- Language switching between English and Spanish
- Synchronized mouth animation with speech
- Modern UI with smooth transitions
- Responsive design
- Easy-to-use interface with text input
- Support for specific voices (Aaron for English, Google español for Spanish)

## Usage

1. Open `index.html` in a modern web browser
2. Select your preferred language (English/Spanish)
3. Type your text in the textarea
4. Click "Speak" to make the avatar talk
5. Click "Stop" to stop the speech at any time

## Technical Details

- Uses Web Speech API for text-to-speech
- CSS animations for mouth movement
- Vanilla JavaScript implementation
- No external dependencies required

## Browser Support

Works best in modern browsers that support:
- Web Speech API
- CSS Transitions
- Modern JavaScript features
